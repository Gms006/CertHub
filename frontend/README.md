# CertHub Frontend

## Desenvolvimento

```bash
npm install
npm run dev
```

### Variáveis de ambiente

Copie `.env.example` para `.env` e ajuste conforme necessário:

```bash
VITE_API_URL=http://localhost:8000/api/v1
```

`VITE_API_URL` (opcional) define a URL base da API. Padrão: `http://localhost:8000/api/v1`.
